# Spotify-Clone-API
 
Functionality:
  Password Hashing:
    Bcrypt
  Authentication/login:
    JSON Web Tokens
  Environment Variables:
    dotenv
  
In this API we can do 
  USER( create, update ), 
  SONG( create, update, delete ), 
  LIKE_SONG( add, remove ), 
  PLAYLIST( create, update, remove )
